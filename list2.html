1. Contents Image ex06/index ● 2017-03-01
Breakpoint
768px, 960px
Contents Image 위에 텍스트를 표시하는 방법
img 에 대체텍스트를 입력하여 스크린 리더기에서 읽어줄 수 있도록 한다.
a 요소에 data-text 라는 속성을 만들고 img의 대체텍스트와 동일한 값 입력
a의 가상요소에 content: attr(data-text) 속성을 주어 텍스트를 표시
이미지 목록을 반응형으로 제작 시 체크사항
퍼센트값으로 넓이를 지정하다 보니 값이 정수로 떨어지지 않는 경우가 발생한다.
이 때 img 요소를 사용하여 이미지 목록을 만들 경우 1px 정도의 세로라인 공백이 간헐적으로 발생하여 디자인 이슈가 우려 된다.
배경 이미지로 처리할 경우엔 가로폭 변동에 자연스럽게 반응하여 공백이 발생하지 않는다.
이슈사항
translateX(-50%) 를 사용하여 텍스트를 가운데 정렬시켰을 때, 텍스트가 좌우로 흔들리는 것이 확인됨
상세내용
a > img 에서 hover, focus 시 a::before, a::after에 transition 효과 적용
opacity: 0 → 1 의 transition 효과 적용
가로폭이 변하지 않는 상태에서도 transition 효과가 끝날 때까지 좌우로 1~2px 씩 흔들림
세로 가운데 정렬은 translateY(-50%)를 사용해도 이상 없음
해결책
동일한 문제 발생시 아래와 같이 코드를 수정한다.
[이슈 CODE] left: 50%; transform: translateX(-50%);
[수정 CODE] left: 0; width: 100%; text-align: center;
