유연한 이미지(Fluid Image)

유연한 이미지(Fluid Image)는 이미지를 포함하는 컨테이너 요소의 폭에 맞춰 크기가 변경되는 이미지를 말한다.

1. Contents Image

  • img에 width: 100%; height: auto; 적용

2. Background Image

padding 과 margin을 퍼센트 값으로 지정할 경우 상위 요소를 기준으로 적용이 되며, 상하 마진/패딩이 퍼센트 값으로 지정될 경우 상위 요소의 너비(width)를 기준으로 적용된다.

  • padding-bottom: (가로 / 세로 * 100)
  • ex) padding-bottom: 100% (가로 : 세로 = 1 : 1)
  • 참고: 마진과 패딩에 퍼센트 값 적용
